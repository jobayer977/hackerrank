## [1929. Concatenation of Array](https://leetcode.com/problems/concatenation-of-array/)

### **Answer:**

```js
/**
 * @param {number[]} nums
 * @return {number[]}
 */
var getConcatenation = function (nums) {
	return nums.concat(nums)
}
```
